# Student Productivity & Digital Distraction Analysis

This is a data analysis of [this](https://www.kaggle.com/datasets/sehaj1104/student-productivity-and-digital-distraction-dataset) dataset, focused to be a learning project in which a narrative is created for better understanding of the data.

## Libraries used

There is, in total, 3 libraries in use for this project:

- Streamlit (Dashboard)
- Plotly (Data Analysis)
- Kagglehub (Dataset Source)

## How to run

Simply download the necessary libraries and run it with the command

``streamlit run main.py``

It should open a web browser link tied to your [localhost on port 8501](http://localhost:8501/)
